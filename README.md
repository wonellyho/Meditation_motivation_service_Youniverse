> ### 핵심 내용
- 사용자로부터 '꿈꾸는 미래', '오늘의 일기'를 입력받는다.
- 각 단계별 입력에 따라 API호출을 활용해 생성 결과물을 받고, 그것을 활용한다.
- 동기, 비동기 방식 2가지를 활용해, 사용자 경험 증대

</br>

- AI와 창의성의 융합 경험 제공
